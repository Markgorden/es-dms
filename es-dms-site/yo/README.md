es-dms-yo
=========

Build process
-------------

Install Node.js [1]  
Install Yo, Grunt and Bower [2]  
```
	npm install -g yo grunt-cli bower
```
Install AngularJS scaffolding  
```
	npm install -g generator-angular
```
Install all dependencies  
From root project folder run:  
```
	npm install
	bower install
```
Run webapp  
```
	grunt server
```
